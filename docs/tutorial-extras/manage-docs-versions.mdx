# Accessibility

This site has many features to make it more accessible.

## Alt Text

The embedded video includes alt text so screen readers can give a description to the user:

```bash
<iframe className="youtube-container" alt="KnowBe4 Holiday Safety Tips" src="https://www.youtube.com/embed/BQ0WrHd3-Jw?si=DLq9tn3xpTM-By-p" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen></iframe>
```

## Navigable by keyboard

Using the tab key allows users to navigate to and open the popup modal.

## Appropriate color schemes

The contrast meets WCAG AAA Accessibility standards in both light and dark mode.
